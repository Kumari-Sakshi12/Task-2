Contact Form & Dynamic To-Do List

## Project Overview

This project is a responsive web application developed using HTML, CSS, and JavaScript. It combines a Contact Form with validation and a Dynamic To-Do List in a single interface. The application allows users to submit contact information and manage daily tasks efficiently.

## Features

Contact Form

- Name, Email, and Message fields
- Required field validation
- Email format validation
- Success message on valid submission

Dynamic To-Do List

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Real-time task management

Task Summary Dashboard

- Total Tasks Counter
- Completed Tasks Counter
- Incomplete Tasks Counter
- Automatic updates when tasks are added, edited, completed, or deleted

 ##Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in any web browser.
4. Start using the Contact Form and To-Do List.
